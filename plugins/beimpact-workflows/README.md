# BE.IMPACT Workflows Plugin

Skills for running BE.IMPACT-specific workflows — matchmaking, impact scoring, and funding roadmaps.

> 🚧 **Coming soon.** Skills in this plugin are under active development.

## Install

```bash
./install.sh beimpact-workflows
```

## Skills planned

| Skill | Description | Status |
|-------|-------------|--------|
| `matchmaking` | Run BE.IMPACT's matchmaking methodology — score compatibility between orgs, investors, and opportunities | 🔜 Planned |
| `impact-scoring` | Apply BE.IMPACT's impact scoring framework to evaluate orgs and opportunities | 🔜 Planned |
| `funding-roadmap` | Generate step-by-step funding roadmaps from current stage to next milestone | 🔜 Planned |
