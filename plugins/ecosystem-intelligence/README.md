# Ecosystem Intelligence Plugin

Skills for navigating the Belgian and European impact ecosystem — funding landscape, investors, accelerators, and ecosystem actors.

## Install

```bash
# Via Claude Code plugin marketplace
claude plugin marketplace add Be-impact/agent-skills
claude plugin install ecosystem-intelligence@beimpact

# Via install script
./install.sh ecosystem-intelligence

# Via ClawHub
npx clawhub install beimpact/ecosystem-intelligence
```

## Skills included

| Skill | Description |
|-------|-------------|
| `belgian-funding-navigator` | Navigate Belgian impact funding — grants, investors, accelerators, EU programs. Personalized funding paths by stage, region, and sector |

## Paste into any AI (no install needed)

Copy the raw URL of any `SKILL.md` and say:
> "Please read this skill and follow its instructions: [URL]"

Raw URLs: `https://raw.githubusercontent.com/Be-impact/agent-skills/main/plugins/ecosystem-intelligence/skills/<skill-name>/SKILL.md`
